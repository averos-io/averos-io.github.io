## **Latest Version**


The latest **Averos** version is `1.3.5` which is compatible with angular version  `<= 13.3.3`.

| **averos version** | **angular version** |
| ------ | ------ |
| `1.3.5` | <= `13.3.3` |
| `1.3.2` | <= `13.2.2` |
| `1.3.0` | <= `13.0.3` |
| `1.2.4` | <= `12.2.6` | 
| `1.2.3` | <= `12.2.2` |
| `1.2.2` | <= `12.2.1` |
